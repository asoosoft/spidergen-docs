# ABar
> **Extends**: `AView`

ABar

<br/>

## Methods

### addToolButton( text )

ABar컴포넌트에 버튼을 추가한다.

* **Parameters**: 
	* **`text`** {String} 버튼에 보여질 text

* **Usage**: 
```js
bar.addToolButton('버튼');
```

<br/>

### init( context, evtListener )

컴포넌트를 생성하고 초기화 할 때 호출한다<br/>동적으로 객체를 생성할 경우 파라미터를 생략하고 호출한다.

* **Parameters**: 
	* **`context`** {String} 컴포넌트 생성 및 초기화 정보
	* **`evtListener`** {String} context 에 매핑된 이벤트 수신자

* **Usage**: 
```js
var bar = new ABar();
bar.init();
```

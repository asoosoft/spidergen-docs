### isHscroll()

### scrollOffset( offset )

툴바의 현 위치에서 offset만큼 Left 스크롤 시킨다

- `offset` \<Number> 이동 시킬 값

<br/>

### scrollTo( leftPos )

툴바의 pos에서 left 위치로 스크롤 시킨다

- `leftPos` \<Number> 이동 시킬 위치 값

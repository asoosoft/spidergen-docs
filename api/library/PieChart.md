## Properties


### centerX

* **Default**: `0`

<br/>

### centerY

Pie차트 중심 Y좌표

* **Type**: `float`
* **Default**: `0`

<br/>

### lineColor

* **Type**: `String`
* **Default**: `'#202023'`

<br/>

### rateColor

* **Default**:
